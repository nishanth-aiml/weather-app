# Weather App 🌤️

A simple, beautiful weather application that shows real-time weather data for any city in the world.

## Features
- 🌡️ Real-time temperature (Celsius)
- 💧 Humidity percentage
- 💨 Wind speed
- 🌡️ "Feels like" temperature
- 🎨 Weather icons (☀️, ☁️, 🌧️, etc.)
- 📱 Responsive design (works on mobile and desktop)

## How to Use
1. Enter a city name in the search box
2. Click "Search" or press Enter
3. View the current weather information

## Technologies Used
- HTML5
- CSS3 (with gradients and animations)
- JavaScript (ES6+)
- OpenWeatherMap API

## How to Get Your Own API Key
1. Go to [OpenWeatherMap](https://openweathermap.org/api)
2. Create a free account
3. Get your API key
4. Replace `YOUR_API_KEY_HERE` in `script.js` with your actual API key
